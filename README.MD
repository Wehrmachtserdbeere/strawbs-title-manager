# Strawb's Title Manager

This is an updated (Nov. 1st, 2025) version of [xman7373's Title Manager](https://steamcommunity.com/sharedfiles/filedetails/?id=2436624088&searchtext=Title+Manager). This is just here to work with Hegemonies, the vast majority of the original mod]was created by xman7373.

This is a mod for Crusader Kings 3. Should it ever stop working, open an issue. This is the non-Steam repository so that those that do not use Steam for mods can use the mod.
